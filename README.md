# 🚀 Portfolio Website

The source code behind my personal developer portfolio

## 📝 About The Project

This repository contains the source code for my personal portfolio website. It is designed to showcase my projects and skills as a Backend Developer & Data Scientist. Built with modern web technologies, it focuses on high performance, accessibility, and a clean user interface.

## ✨ Key Features

* **🚀 Blazing Fast Performance:** Statically generated using Astro, delivering zero-JS by default for maximum speed.
* **📄 Markdown-Driven Content:** Projects and portfolio items are easily managed using Markdown (`.md`) files. No database configuration required.
* **🌗 Dark/Light Mode:** Fully integrated theme toggling that respects system preferences.
* **🎨 Responsive Design:** Mobile-first, fluid, and beautifully styled with Tailwind CSS.
* **✨ Smooth Animations:** Delightful micro-interactions and scroll reveals powered by `motion`.
* **🔍 SEO Optimized:** Includes built-in SEO tags, automated sitemaps, and RSS feed generation.

## 🛠️ Built With

This portfolio leverages the following tech stack:

* [**Astro**](https://astro.build/) - The web framework for content-driven websites.
* [**Tailwind CSS**](https://tailwindcss.com/) - A utility-first CSS framework for rapid UI development.
* [**TypeScript**](https://www.typescriptlang.org/) - For type-safe code and a better developer experience.
* [**Motion**](https://motion.dev/) - For fluid and declarative animations.

## 🏃 Getting Started

To get a local copy up and running, follow these simple steps. This project uses [pnpm](https://pnpm.io/) as the primary package manager.

### Prerequisites

Make sure you have Node.js and `pnpm` installed on your machine.

### Installation

1. **Clone the repository:**

   ```bash
   git clonehttps://github.com/ahmaadn/portfolio.git
   cd portfolio
   ```

2. **Install dependencies:**

   ```bash
   pnpm install
   ```

3. **Run the development server:**

   ```bash
   pnpm run dev
   ```

   Open `http://localhost:4321` in your browser to view the project in action.

## 📂 Managing Content (Projects)

To add or edit your portfolio projects, simply navigate to the `src/content/projects/` directory.

Create a new Markdown (`.md`) file, fill in the frontmatter details (such as title, description, and cover image), and it will automatically be rendered on the website.

## 📜 Available Scripts

| Command | Description |
| :--- | :--- |
| `pnpm run dev` | Starts the local development server (with HMR) |
| `pnpm run build` | Builds the static application for production deployment |
| `pnpm run preview` | Previews the production build locally |
| `pnpm run format` | Formats the source code using Prettier |
